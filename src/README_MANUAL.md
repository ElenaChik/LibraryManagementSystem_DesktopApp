<h1>Library Management Application</h1>
<p>
This application provides functionality to easily manage a Library. <br/>
It has two types of users: Librarian and Student,<br/>
with different interfaces and functionality for each one.  <br/><br/>
for LIBRARIAN:<br/>
<ul>
<li>Add Users;</li><br/>
<li>Add books;</li><br/>
<li>Update books quantity;</li><br/>
<li>See "Books" Catalog</li><br/>
<li>See "Available Books" Catalog;</li><br/>
<li>See "Issued Books" Catalog;</li><br/>
<li>Filter all Catalogs;</li><br/>
<li>Sort Catalogs by any field;</li><br/>
<li>Issue Books;</li><br/>
<li>Return Books;</li><br/>
<li>Change Language;</li><br/>
<li>See Books price in different currency<br/><br/>
according to the selected language;</li>
</ul>

<br/><br/>
for USER:<br/>
<ul>
<li>See "All Books" Catalog;</li><br/>
<li>See "Available Books" Catalog;</li><br/>
<li>Filter and Sort Catalogs;</li><br/>
<li>Borrow Book from Available;</li><br/>
<li>See All "Borrowed Books";</li><br/>
<li>Return Books;</li>
</ul>

<br/><br/>
This Role based application has:<br/><br/>
<li>Users Authentication;</li><br/>
<li>Encrypted passwords;</li><br/>
<li>Verification of ISBN validity;</li><br/>
<li>Informative Error Messages;</li><br/>
</ul><br/><br/>
